description

### Setup
``giy clone git@git.cdwv.pl:cdwv/flotiq-cli.git``

`` cd flotiq-cli``

`` npm install``


### Usage
Run:

``` node index.json start [apiKey] [projectName] [flotiqSterterUrl]```

Or for load examples:
 
``` node index.json import [apiKey] [projectName]```
